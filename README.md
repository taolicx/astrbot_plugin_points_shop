# AstrBot Points Shop

群聊积分兑换插件，支持：

- 群聊直接发送 `签到` 获取积分
- 群聊直接发送 `猜拳 石头 10` 进行积分下注
- 所有群聊共享同一套积分、签到、兑换记录
- 商品兑换与精美海报展示
- `reward_mode = pool` 商品自动私发兑换码
- 插件页面 `code_manager` 专门管理兑换码仓库

当前版本：`0.1.8`

## 功能说明

### 1. 群聊积分

- `签到`：每日签到获取积分
- `积分`：查看余额
- `积分排行`：查看全局排行

### 2. 剪刀石头布

- `猜拳 石头 10`
- 支持配置最小下注、最大下注
- 支持配置玩家获胜概率 `rps_win_rate`
- 支持配置平局概率 `rps_draw_rate`
- 赢了返还双倍，输了不返还，平局退回本金

### 3. 商品兑换

- `商店`：查看商品海报
- `兑换 <商品ID或名称> [数量]`
- `兑换记录`：查看最近订单

商品支持两种发货模式：

- `manual`：群内提示兑换成功，管理员手动核销
- `pool`：从兑换码仓库扣减，并通过私聊自动发送兑换码

## 兑换码仓库

从 `0.1.6` 开始，仓库奖励统一为“兑换码”模式，不再支持图片奖励。

### 推荐管理方式

在 AstrBot 插件页面打开：

- `code_manager`

页面功能：

- 选择 `reward_mode = pool` 的商品
- 批量添加兑换码（每行一个）
- 查看当前余量
- 按关键词筛选
- 删除单条兑换码
- 一键清空当前商品兑换码

### 备用群聊指令

管理员仍可用群聊命令紧急补码：

```text
奖励入库 mystery ABCD-EFGH-IJKL
奖励入库 mystery ABCD-EFGH-IJKL | 6月活动批次
奖励仓库 mystery
```

用户兑换 `pool` 商品后，群里只提示“奖励已私发”，实际兑换码通过私聊发送。

## 商品配置

在插件配置里的 `items` 中填写商品：

```json
{
  "id": "mystery",
  "name": "神秘盲盒",
  "price": 300,
  "stock": 3,
  "emoji": "🎁",
  "description": "随机小奖励，开盒有惊喜",
  "delivery": "奖励会通过私聊自动发放。",
  "color": "#db2777",
  "reward_mode": "pool",
  "enabled": true
}
```

字段说明：

- `id`：兑换时使用的商品 ID
- `name`：商品名称
- `price`：兑换所需积分
- `stock`：库存，`-1` 表示无限
- `emoji` / `description` / `color`：商品展示信息
- `delivery`：`manual` 模式下显示给用户的说明
- `reward_mode`：`manual` 或 `pool`
- `enabled`：是否启用商品

## 常用指令

```text

```

## 安装

1. 将插件目录放入 AstrBot 插件目录
2. 安装依赖：

```bash
pip install -r requirements.txt
```

3. 在 AstrBot 面板启用插件
4. 配置 `items`
5. 如果有自动私发商品，把对应商品 `reward_mode` 设为 `pool`
6. 打开插件页面 `code_manager` 批量导入兑换码
