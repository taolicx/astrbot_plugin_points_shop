# AstrBot Points Shop

一个面向群聊的积分兑换插件，支持签到、猜拳下注、积分商城、奖励仓库私发，以及精美商品展示图。

## 功能概览

- 群聊里直接发送 `签到`、`积分`、`商店` 等纯文字即可使用，也兼容 `/签到` 这类前缀命令
- 所有群聊共享同一套积分余额、签到状态、兑换记录
- 每日签到获取积分，支持连续签到额外奖励
- 与机器人玩剪刀石头布下注，最小/最大押注、玩家胜率、平局率都可配置
- 商店可生成精美海报图片；未安装 Pillow 时自动回退为文字列表
- 商品支持两种发货模式：
  - `manual`：群内提示兑换成功，由管理员手动核销
  - `pool`：从奖励仓库自动取货，群内提示“奖励已私发”，奖励内容通过私聊发送
- 奖励仓库支持入库文字、兑换码、链接、图片、本地图片路径、base64 图片等内容

## 常用指令

- `签到`：每日签到领取积分
- `积分`：查看当前积分余额
- `积分排行`：查看全局积分排行榜
- `猜拳 石头 10`：押 10 积分和机器人猜拳
- `商店`：查看商品海报
- `兑换 mystery 1`：兑换指定商品
- `兑换记录`：查看最近订单
- `积分帮助`：查看帮助

管理员额外指令：

- `积分管理 123456 +50`
- `奖励入库 mystery 文本 ABCD-EFGH-IJKL | 第1批兑换码`
- `奖励入库 mystery 图片 https://example.com/code.png | 二维码奖励`
- `奖励仓库 mystery`

`奖励入库` 在图片模式下也支持直接附带一张图片发送，例如：

```text
奖励入库 mystery 图片 | 微信兑换二维码
```

然后消息里附上一张图片即可。

## 猜拳规则

- 下注前会先扣除对应积分
- 赢：返还双倍积分，相当于净赚一倍押注
- 输：不返还押注
- 平：返还本金

相关配置项：

- `min_bet`：最小押注
- `max_bet`：最大押注
- `rps_win_rate`：玩家获胜概率，范围 `0-100`
- `rps_draw_rate`：平局概率，范围 `0-100`

剩余概率自动视为玩家失败概率。如果胜率和平局率相加超过 `100`，插件会自动压低平局率。

## 商品配置

在 AstrBot 插件配置中修改 `items`。每个商品支持以下字段：

```json
{
  "id": "mystery",
  "name": "神秘盲盒",
  "price": 300,
  "stock": 3,
  "emoji": "🎁",
  "description": "随机小奖励，开盒有惊喜",
  "delivery": "奖励会通过私聊自动发放。",
  "color": "#db2777",
  "reward_mode": "pool",
  "enabled": true
}
```

字段说明：

- `id`：兑换时使用的商品 ID
- `price`：单件兑换价格
- `stock`：库存；`-1` 表示无限库存
- `emoji` / `description` / `color`：用于商店海报展示
- `delivery`：`manual` 模式下给用户的兑换说明
- `reward_mode`：发货模式，支持 `manual` 和 `pool`
- `enabled`：是否启用该商品

## 奖励仓库说明

当商品设置为 `reward_mode = "pool"` 时：

1. 管理员先用 `奖励入库` 往对应商品仓库中添加奖励
2. 用户在群聊里兑换该商品
3. 插件自动从仓库中取出对应数量的奖励
4. 群里只显示“奖励已私发，请查收私聊”
5. 奖励内容通过私聊发送给兑换者

如果当前平台不支持主动私聊，或私聊发送失败，群里会提示联系管理员补发。

## 安装

1. 将 `astrbot_plugin_points_shop` 放入 AstrBot 插件目录
2. 安装依赖：`pip install -r requirements.txt`
3. 在 AstrBot 面板启用插件
4. 按需调整插件配置
5. 在群聊中发送 `积分帮助` 开始使用
