# AstrBot 积分兑换商城

一个群聊积分兑换插件：用户可以签到获得积分，也可以在群聊中和机器人玩剪刀石头布下注。积分可兑换插件配置里的商品，发送 `/商店` 会生成一张动态商品展示图。

## 功能

- 每日签到：`/签到`
- 查看积分：`/积分`
- 群积分排行：`/积分排行`
- 猜拳下注：`/猜拳 石头 10`
- 商品海报：`/商店`
- 积分兑换：`/兑换 coffee 1`
- 兑换记录：`/兑换记录`
- 管理员调整积分：`/积分管理 用户ID +50`

## 猜拳规则

下注前会先扣除对应积分。

- 赢：获得下注积分的双倍返还，等于净赚下注额。
- 输：下注积分不返还。
- 平：返还本金。

最小下注和最大下注可在插件设置里调整：`min_bet`、`max_bet`。

获胜概率可在插件设置里调整：

- `rps_win_rate`：玩家获胜概率，默认 `33`，表示 33%。
- `rps_draw_rate`：平局概率，默认 `33`，剩余概率自动作为玩家失败概率。

例如 `rps_win_rate=50`、`rps_draw_rate=20` 时，玩家约 50% 获胜、20% 平局、30% 失败。

## 商品配置

在 AstrBot 插件设置中修改 `items`。每个商品支持这些字段：

```json
{
  "id": "coffee",
  "name": "今日咖啡",
  "price": 60,
  "stock": -1,
  "emoji": "☕",
  "description": "给自己兑换一杯精神补给",
  "delivery": "请联系管理员核销。",
  "color": "#6f4e37",
  "enabled": true
}
```

说明：

- `id` 是兑换时使用的短 ID。
- `price` 是兑换单价。
- `stock` 为 `-1` 表示无限库存，其他数字表示剩余库存。
- `emoji`、`description`、`color` 会显示在商城图片上。
- `delivery` 会在兑换成功后作为核销说明回复。

## 安装

1. 将 `astrbot_plugin_points_shop` 文件夹放入 AstrBot 插件目录。
2. 安装依赖：`pip install -r requirements.txt`
3. 在 AstrBot 面板启用插件。
4. 在群聊发送 `/积分帮助` 查看指令。

如果没有安装 Pillow，插件仍会回退为文字商品列表，但 `/商店` 的精美图片需要 Pillow。
